# QvPlugin-SS-NOAVX


适用于 Qv2ray 的 ShadowSocks 插件，使用此插件在 Qv2ray 中启用 SIP003 完整支持

相对于原版调整了编译指令使其可在不支持AVX指令的旧CPU上运行

仅在Ubuntu20.04LTS进行测试
